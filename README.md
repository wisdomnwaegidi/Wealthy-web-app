# Setting Up the Wealthy home-web-app

This guide will walk you through the process of setting up the Wealthy Homes Web App on your local machine.

## Prerequisites

Before you begin, ensure you have Node.js installed on your system.

## Cloning the Repository

Start by cloning the repository to your local machine:

```bash
git clone https://github.com/wisdomnwaegidi.git
cd Wealthyhome-web-app
```
